# Отчет по проекту "Интернет магазин" по дисциплине "Проектный практикум"

Роль в проекте: Backend-разработчик

Основная обязанность: реализовать и протестировать модули для работы с корзиной и каталогом лдя пользователей

## Перечень выполненных задач

1) Разработка модуля корзины (cart)

   - Задача: Реализация логики работы с корзиной покупок
   - Выполненные работы:
     - Создание моделей SQLAlchemy для корзины и товаров в корзине
     - Разработка CRUD операций для управления корзиной
     - Настройка зависимостей между пользователем и его корзиной
     - Реализация проверок прав доступа (пользователь может работать только со своей корзиной)
     - Реализация необходимых эндпоинтов:
       - GET /api/v1/cart/
       - POST /api/v1/cart/items
       - PUT /api/v1/cart/items/{product_id} и другие
   - Разработанный модуль был интегрирован в проект и используется другими модулями, реализованными другими участниками проектв
2) Разработка модуля каталог (catalog)
   - Задача: Создание логики работы с товарами и категориями
   - Выполненные работы:
     - Реализация CRUD операций для товаров
     - Создание публичных эндпоинтов для получения информации о товарах
     - Разработка системы категорий товаров
     - Реализация административных эндпоинтов для управления каталогом и категориями
     - Реализация необходимых эндпоинтов для работы:
       - GET /api/v1/catalog/products
       - POST /api/v1/catalog/products
       - PUT /api/v1/calatog/categories/{category_id} и другие

Данные модули были реализованны и успешно протестированны с использованием pytest и aiohttp

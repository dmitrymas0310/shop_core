# Отчет по проекту "Интернет магазин" по дисциплине "Проектный практикум"

Роль в проекте: Backend-разработчик

Основная обязанность: разработка и сопровождение модуля Promotions

## Перечень выполненных задач

1) Разработка и стабилизация модуля Promotions

   - Задача: Реализация логики акций и связей с товарами
   - Выполненные работы:
     - Настройка API-эндпоинтов для управления акциями
     - Настройка доступа к административным операциям модуля
     - Обеспечение корректных статус-кодов (201 при создании акции)
     - Доработка работы с привязкой/отвязкой товаров к акциям
     - Улучшение получения данных по акции с учетом связей с товарами

2) Трудности в работе

   - Конфликты миграций Alembic
   - Состояние БД после обновления схемы
   - Зависимость тестов от внешнего запуска сервера

Данные изменения обеспечили корректную работу модуля Promotions и его интеграцию с остальными частями проекта. Тесты модуля были проверены и проходят.

3) Используемый стек технологий

- Python 3.12
- FastAPI
- Poetry
- Uvicorn
- PostgreSQL
- SQLAlchemy (async)
- Pydantic
- Alembic
- Docker
- Pytest
- git

4) Саморефлексия

 Достижения:
    - стабилизировал работу модуля Promotions
    - улучшил админскую авторизацию и корректность ответов API
    - закрепил опыт работы с асинхронным стеком и миграциями

 Трудности:
    - конфликты миграций Alembic
    - состояние БД при обновлениях схемы
    - зависимость тестов от внешнего запуска сервера

 Впечатления от командной работы:
    - совместное решение проблем ускорило отладку
    - анализ чужого кода помог быстрее выявлять несовместимости
